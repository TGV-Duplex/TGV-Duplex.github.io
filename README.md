# **0. 前言**
---

>  工业机器人是广泛用于工业领域的多关节机械手或多自由度的机器装置，具有一定的自动性，可依靠自身的动力能源和控制能力实现各种工业加工制造功能。工业机器人被广泛应用于电子、物流、化工等各个工业领域之中。\
>  一般来说，工业机器人由三大部分六个子系统组成。\
> 三大部分是机械部分、传感部分和控制部分。\
> 六个子系统可分为机械结构系统、驱动系统、感知系统、机器人-环境交互系统、人机交互系统和控制系统。\
> <p align="right">——摘自《百度百科》</p>
